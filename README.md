# SCUM Raid Explosive Calculator

A Vue.js web application for calculating explosive requirements to raid bases in the SCUM game.

## Features

- **Explosive Calculator**: Calculate exact explosive requirements for different building elements
- **Multi-language Support**: 12 languages included with translation interface
- **Dark/Light Theme**: Toggle between themes
- **Damage Configuration**: Customize game balance data
- **Translation Tool**: Built-in interface for contributing translations

## Available Languages

- English (en)
- Croatian (hr)
- German (de)
- Spanish (es)
- French (fr)
- Italian (it)
- Korean (ko)
- Dutch (nl)
- Russian (ru)
- Slovenian (sl)
- Swedish (sv)
- Chinese (zh)

## Technology Stack

- Vue.js 3
- Vite
- Vue I18n for internationalization
- Font Awesome icons
- CSS3 with custom themes
